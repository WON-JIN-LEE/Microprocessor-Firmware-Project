# 마이크로 프로세서

### Project

- 개발인원 : 2명
- 개발 툴 : AVR Studio4
- 언어 : C
- 개발 기간: 2019.05.01. ~ 2019.05.24

### 주요 부품:

1. 마이크로 프로세서(Atmega128)
2. 초음파센서( HC-SR04)
3. 블루투스 (HC-05)

### 주요기능

- App과 디바이스간에 블루투스 통신으로 제어할 수 있습니다.
- 선풍기 ON 상태이면 초음파 센서에 잡히는 대상에게 선풍기가 자동으로 회전할 수 있게 구현하였습니다.
